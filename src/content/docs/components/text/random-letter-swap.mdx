There are two types of animations available for this component:

1. Forward animation — plays the animation timeline once forward.
2. Ping Pong animation — plays the animation timeline in a ping pong fashion.

## Understanding the component

The component works the same as the [Letter Swap Hover](/docs/components/text/letter-swap) component, but with a random letter swapping animation (and of course, a slightly different implementation of the animation). Please refer to that documentation for that component for more details.
